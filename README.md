# SMP

1. Financial - Business Reality
   - Income Statement - earning (company's actual earnings)
   - Balance Sheet - stability (assets, liabilities and debt)
   - Cash Flow - real cash analysis (cash flow activities and financial dependency)
3. Fundamentals - Business Strength
4. Performance - Market Behaviour
5. Technical - Timing Signals
